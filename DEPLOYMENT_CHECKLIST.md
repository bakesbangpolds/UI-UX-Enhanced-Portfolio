## Quick Start Guide

### 1. GitHub Repository Setup

Your GitHub repository is already initialized at:
```
https://github.com/r4diorusak/UI-UX-Enhanced-Portfolio
```

### 2. Enable GitHub Pages

Follow these steps to deploy:

#### Option A: Automatic (Recommended)
1. Go to repository **Settings**
2. Navigate to **Pages** section (left sidebar)
3. Under "Source", select **Branch: main**
4. Select folder: **/ (root)**
5. Click **Save**
6. Wait 1-2 minutes for deployment

#### Option B: Using gh-pages Branch
1. Create or checkout `gh-pages` branch
2. Push all files to this branch
3. Set GitHub Pages source to `gh-pages` branch

- ✅ Check repository "About" section for live URL

## Troubleshooting

### Site not loading after GitHub Pages setup?
1. Wait 2-3 minutes for initial deployment
2. Hard refresh browser (Ctrl+Shift+R)
3. Check Settings → Pages for deployment status
4. Verify branch is set to `main` and folder is `/`

### Styling not showing?
1. Verify `styles.css` is linked in `index.html`
2. Check browser console for CSS file 404 errors
3. Ensure file paths are relative (e.g., `styles.css` not `/styles.css`)

### JavaScript not working?
1. Verify `script.js` is linked at end of HTML
2. Check browser console for errors
3. Ensure IDs in HTML match JavaScript selectors

### Mobile menu not responsive?
1. Test viewport with DevTools mobile emulation
2. Verify media query breakpoints in CSS
3. Check hamburger button is clickable
