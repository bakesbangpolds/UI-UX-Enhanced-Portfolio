# UI-UX Enhanced Portfolio

A comprehensive portfolio website for Alex Styles, developed as a Coursera assignment combining HTML, CSS, and interactive UI/UX design principles. This project demonstrates the integration of Microsoft Copilot assistance throughout the development process.

## Project Overview

This portfolio website showcases a professional online presence with:
- **Semantic HTML** structure for accessibility and SEO
- **Responsive Design** that adapts to all screen sizes (mobile, tablet, desktop)
- **Interactive Features** including hover effects, modals, and tooltips
- **Visual Enhancements** with modern design aesthetics
- **Professional Styling** using modern CSS practices

## Features

### Core Sections
- **Header & Navigation** - Clean navigation bar with smooth scrolling
- **Hero Section** - Eye-catching introduction with call-to-action
- **Portfolio Projects** - Showcase of work with project cards
- **About Section** - Professional background and skills
- **Contact Section** - Contact information and inquiry form
- **Footer** - Links and copyright information

### Interactive Elements
- Smooth scroll navigation
- Hover effects on portfolio items
- Interactive tooltips for skills
- Modal dialogs for project details
- Responsive hamburger menu for mobile
- Form validation and feedback

### Design Features
- Modern color scheme with gradient accents
- Professional typography
- Smooth animations and transitions
- Box shadows for depth
- Consistent spacing and alignment

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling, animations, and responsive design
- **JavaScript** - Interactive features and DOM manipulation
- **GitHub Pages** - Hosting and deployment
